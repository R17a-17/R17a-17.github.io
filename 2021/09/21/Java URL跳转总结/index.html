URL跳转, URL重定向

URL跳转和URL重定向的区别：

  1. 地址栏变化：URL跳转地址栏不变，重定向地址栏会变；
  2. 服务端还是浏览器端跳转：URL跳转是服务端跳转，重定向是客户端跳转
